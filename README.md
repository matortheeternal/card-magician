# card-magician

## how to run locally

1. clone the repository with git or a git client.
2. initialize submodules with `git submodule update --init`
3. install dependencies: NodeJS and npm
4. run `npm i` in project directory
5. run `npm install -g @neutralinojs/neu`
6. run `neu update` to install neutralino binaries
7. run `npm start`
